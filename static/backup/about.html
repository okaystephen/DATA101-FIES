About this example

This mini demo shows how we can turn our initial JSBin example to a hosted application on Flask. The static HTML, CSS and JavaScript files no longer have the C:/.../index.html address in the browser's address bar, but instead, it's hosted on the localhost server by Flask.

It also shows you how you can create specific data request endpoints that only provide you filtered data, or even processed data (depending on your needs).
